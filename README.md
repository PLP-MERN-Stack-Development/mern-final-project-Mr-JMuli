# SkyPharma ☁️ – Intelligent Online Pharmacy (MERN Capstone)


## Features
- Full user/auth system with JWT & roles
- Prescription upload + pharmacist approval workflow
- Medicine catalog with Rx check
- Real-time order tracking with Socket.io
- Responsive Tailwind UI
- Pharmacist dashboard

## Tech Stack
MongoDB • Express • React (Vite) • Node.js • Socket.io • Tailwind • Cloudinary • React Query

## Quick Start
```bash
git clone 
cd backend && npm install && npm run dev
cd frontend && npm install && npm run dev
